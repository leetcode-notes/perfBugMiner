Update CUDA capability to 3.5 when install from binary in documentation

This fix tries to address the issue raised in #17445. The minimal CUDA capability is 3.5 for Linux install from binary, though the documentation only specifies 3.0.
This fix updates the docs to 3.5 for binary install.
This fix fixes #17445.
